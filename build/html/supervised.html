supervised.classification module¶

class supervised.classification.Classifier(model, norm_func=None, k=0)¶

Bases: object

A custom classifier that uses a specified model and a normalization function to classify data.

Parameters:

  • model (object) – The model to be used for classification. Must have a .fit method.

  • norm_func (callable, optional) – A function that computes a norm of the input data. Default is L2 norm.

  • k (int, optional) – A parameter used to determine the threshold for selecting extreme points. If k=0, it is automatically set to 4 * sqrt(n_samples).

supervised.tirex module¶

supervised.tirex.tirex_transform(X, y, n_components, k=None, method='FO', mode='TIREX', get_SDR_X=False)¶

Perform the TIREX transformation on the input data.

Parameters:

  • X (array-like of shape (n_samples, n_features)) – The input data.

  • y (array-like of shape (n_samples,)) – The target values.

  • n_components (int) – The number of components to extract.

  • k (int, optional) – The number of samples to consider. If None, it is set to the number of samples in X.

  • method (str, optional) – The method to use for transformation. Must be “FO” or “SO”.

  • mode (str, optional) – The mode to use for transformation. Must be “TIREX” or “CUME”.

  • get_SDR_X (bool, optional) – Whether to return the extreme space in the original feature space.

Returns:

  • extreme_space (array-like of shape (n_features, n_components)) – The extreme space components.

  • extreme_space_X (array-like of shape (n_features, n_components), optional) – The extreme space components in the original feature space, if get_SDR_X is True.
