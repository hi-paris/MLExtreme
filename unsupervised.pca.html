class unsupervised.pca.pca.Xpca(thresh_train=None, thresh_predict=None, beta=1, centered=False)[source]¶

Bases: object

A class for performing Principal Component Analysis (PCA) on extreme points of a dataset.

This class provides methods to fit the PCA model to training data, compute scores for test data, predict the projection of test data onto the principal axes, generate scree plots, and calculate reconstruction errors.

fit(X, max_dim=None)[source]¶

Perform PCA by computing the singular value decomposition of the rescaled extreme points on the training dataset X.

Each extreme point x is rescaled by the factor ||x||^{-beta}. The default value for beta is 1. Admissible values are in the interval (1 - alpha/2, 1], where alpha is the regular variation index of the input X.

recons_error(X_test, max_dim, test_beta=None)[source]¶

Calculate the reconstruction error for the test data.

The test data are rescaled to ensure existence of second moments, namely each test data is transformed following z = ||xtest||^{-beta} xtest, where beta <= 1 is given by the argument test_beta. Default is test_beta=None, in which case test_beta is set to the training value self.beta which defaults to 1.

scores(Xtest, max_dim=None)[source]¶

Return the scores, i.e., the coordinates of the points in the array Xtest, in the coordinate system given by the principal axes of the extreme points (with norm greater than self.thresh_predict) rescaled through the mapping x –> ||x||^{-beta} * x where beta = self.beta. The model must have been fit beforehand. If Xtest is the training data, this gives the usual PCA scores of the rescaled extreme data.
